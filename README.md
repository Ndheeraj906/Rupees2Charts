# Rupees2Charts 📊

A clean and simple monthly income & expense tracker with a glassmorphic design and real-time charts.

🚀 **Live Demo:** [https://rupees2charts-478532845626.us-central1.run.app](https://rupees2charts-478532845626.us-central1.run.app)

---

## ✨ Features
- **Dashboard**: Track income and expenses across 12 months.
- **Visualization**: Real-time bar charts comparing monthly data (₹).
- **Secure**: Personal accounts with encrypted login/registration.
- **Responsive**: Works perfectly on mobile and desktop.

---

## 🛠 Tech Stack
- **Frontend**: HTML, Vanilla CSS, Chart.js
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Neon)
- **Deployment**: Google Cloud Run

---

## 🚀 Local Setup
1. **Clone & Setup**:
   ```bash
   git clone https://github.com/Ndheeraj906/Rupees2Charts.git
   cd Rupees2Charts
   cp .env.example .env
   ```
2. **Run with Docker**:
   ```bash
   docker compose up --build
   ```
Access at `http://localhost:3000`.
